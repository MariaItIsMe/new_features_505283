# Математические формулы
Библиотека для языка программирования Python, которая умеет находить площадь и периметр четырех фигур: круг, прямоугольник, квадрат, треугольник.
## Площадь
### Круг
Для круга данная библиотека будет использовать всем известную формулу нахождения площади круга:
- Круг: S = πR²
Чтобы воспользоваться данной функцией нужно прописать def area(r), где r - радиус круга, возвращаемое значение - площадь круга
Разберем на простом примере:
r = 1, def area(r) вернет нам значением 3.141592653589793
### Прямоугольник
Для прямоугольника данная библиотека будет использовать всем известную формулу нахождения площади прямоугольника:
- Прямоугольник: S = ab
Чтобы воспользоваться данной функцией нужно прописать def area(a,b), где a и b - стороны прямоугольника, возвращаемое значение - площадь прямоугольника
Разберем на простом примере:
a = 10,b = 5 def area(a,b) вернет нам значением 50  
### Квадрат
Для квадрата данная библиотека будет использовать всем известную формулу нахождения площади квадрата:
- Квадрат: S = a²
Чтобы воспользоваться данной функцией нужно прописать def area(a), где a - сторона квадрата, возвращаемое значение - площадь квадрата
Разберем на простом примере:
a = 2, def area(a) вернет нам значением 4
### Треугольник
Для треугольника данная библиотека будет использовать всем известную формулу нахождения площади треугольника:
- Треугольник: S = (a × h) / 2
Чтобы воспользоваться данной функцией нужно прописать def area(a,h), где a - основание треугольника, h - высота треугольника, возвращаемое значение - площадь треугольника
Разберем на простом примере:
a = 10, h = 5 def area(a,h) вернет нам значением 25.0
## Периметр
### Круг
Для круга данная библиотека будет использовать всем известную формулу нахождения периметра круга:
- Круг: P = 2πR
Чтобы воспользоваться данной функцией нужно прописать def perimeter(r), где r - радиус круга, возвращаемое значение - длина окружности
Разберем на простом примере:
r = 1, def perimeter(r) вернет нам значением 6.283185307179586
### Прямоугольник
Для прямоугольника данная библиотека будет использовать всем известную формулу нахождения периметра прямоугольника:
- Прямоугольник: P = 2a + 2b
Чтобы воспользоваться данной функцией нужно прописать def perimeter(a,b), где a и b - стороны прямоугольника, возвращаемое значение - периметр прямоугольника
Разберем на простом примере:
a = 3,b = 4 def perimeter(a,b) вернет нам значением 14
### Квадрат
Для квадрата данная библиотека будет использовать всем известную формулу нахождения периметра квадрата:
- Квадрат: P = 4a
Чтобы воспользоваться данной функцией нужно прописать def perimeter(a), где a - сторона квадрата, возвращаемое значение - периметр квадрата
Разберем на простом примере:
a = 2, def perimeter(a) вернет нам значением 8
### Треугольник
Для треугольника данная библиотека будет использовать всем известную формулу нахождения периметра треугольника:
- Треугольник: P = a + b + c
Чтобы воспользоваться данной функцией нужно прописать def perimeter(a,b,c), где a, b, c - стороны треугольника, возвращаемое значение - периметр треугольника
Разберем на простом примере:
a = 1, b = 1, c = 1 def perimeter(a,b,c) вернет нам значением 3
